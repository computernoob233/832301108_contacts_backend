# Code Style Guide

This project follows official standards recommended by major companies.

## Python Code Style
- Based on [PEP 8 -- Style Guide for Python Code](https://peps.python.org/pep-0008/)
- Indentation: 4 spaces
- Line length: max 79 characters
- Use descriptive variable names
- Write docstrings for functions and classes

## JavaScript Style (Frontend)
- Based on [Google HTML/CSS Style Guide](https://google.github.io/styleguide/htmlcssguide.html)
- Use camelCase for variables
- Indent with 2 spaces
- Avoid global variables
